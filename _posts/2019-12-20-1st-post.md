## Dip Trace
Hot keys

space or r  : rotate components  
Ctrl + 2    : Design manager  
Ctrl + m    : copy matrix  
F11         : grin on/off  
F10         : Move part markings mode  

pattern preview에 pattern이 나오지 않는 component는 적절한 pattern을 component symbol에 붙여줘야  
Circuit board에서 error가 발생하지 않는다.
